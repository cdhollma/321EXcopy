You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1={number to multiply}&num2={number to multiply} -- Returns the product of num1 and num2
    • Example: http://35.85.218.115:9000/multiply?num1=10&num2=12
  • /github?query={api call to github api in the form of: users/USERNAME/repos -- Returns information about all repos associated with a given username
    • Example: http://35.85.218.115:9000/github?query=users/amehlhase316/repos
  • /pokemon?poke={Pokemon name/id}&xp={amount of xp needed until next level} -- returns if defeating the given Pokemon would give the player a level up based on given xp amount
    • Example 1: http://35.85.218.115:9000/pokemon?poke=mew&xp=200
    • Example 2: http://35.85.218.115:9000/pokemon?poke=bulbasaur&xp=100
  • /count?str={input string}&target={target character}
    • Example: http://35.85.218.115:9000/count?str=Hello World!&target=l
